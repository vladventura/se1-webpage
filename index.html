Project Background

The Blackboard’s calendar feature seems very limited. We think this part of the blackboard can be improved. There are features in blackboard calendar that is not available in many common calendar applications. For example, blackboard calendar does not allow its users to send an event to different calendar. It also doesn’t have important features such as setting up a repeating event and event notifications. We think we can do better and create a feature rich calendar which has more use to students than just viewing assignment due dates in calendar form.

Brief Description

There are some features that would make the calendar application within blackboard much more useful to a student. We are introducing improved and brand-new features to the Blackboard’s calendar application which will improve a student’s ability to track assignments and their progress in class.

  1. The user can set up repeating events
  2. An event can be transferred to a different personal calendar
  3. Notifications can be set for events – such as a three-day warning before an exam
  4. Options such as hiding completed events or have a check mark display on turned in assignments
